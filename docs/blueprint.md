# **App Name**: Aether Panel

## Core Features:

- Secure Authentication & Authorization: Implement distinct login flows and access control for 'admin' and 'user' roles to restrict feature visibility and functionality based on privileges.
- Server Overview Dashboard: A dashboard displaying a list of all servers (for admin) or assigned servers (for user), showing essential real-time status indicators and basic metrics.
- Server Management (Admin): For 'admin' users to add new servers, modify existing server configurations, and remove servers from the system.
- User Management (Admin): For 'admin' users to create new user accounts, assign roles (admin/user), and manage which servers each 'user' account can access.
- Server Details & Metrics Display: View a detailed page for each server, showing more in-depth real-time performance metrics and status information.
- AI Server Alert Summaries: An AI tool that takes complex server logs or alert messages and generates concise, easy-to-understand natural language summaries for administrators.

## Style Guidelines:

- A strong, professional blue (#2563EB) used for primary buttons, links, and key interactive elements, standing out clearly against dark backgrounds while conveying reliability and modern technology.
- A deep blue-grey (#0F172A) used as the main background, providing a sleek, modern dark canvas that reduces eye strain and enhances focus on content.
- A vivid blue (#3B82F6) to highlight secondary actions, hover states, active elements, and important UI feedback, ensuring clear visual hierarchy and contrast in dark mode.
- A vibrant sky blue (#0EA5E9) for main interactive elements and branding, delivering a fresh and energetic contrast against dark interfaces.
- A dark navy-blue tone (#020617) that creates depth and a premium dark appearance while maintaining excellent readability.
- A deeper sky blue (#0284C7) used for call-to-action buttons, active states, indicators, and highlighted information, adding clarity and emphasis without overwhelming the UI.
- Font family: 'Inter' (sans-serif) for all text elements. Its modern, clear, and objective design ensures excellent readability for both headlines and data-heavy content in the admin panel.
- Use clear, vectorized icons for navigation, server statuses, and management actions (e.g., add server, delete user), ensuring they are universally understandable and visually consistent.
- Implement a clean, organized layout with a clear sidebar navigation for different sections (Admin vs. User areas) and dashboards featuring well-structured tables and charts for data display, optimized for responsive viewing on various screen sizes.
- Subtle and functional animations for UI feedback, such as hover effects on interactive elements, smooth transitions during data loading, or discreet highlights on alert notifications to maintain a polished user experience without distractions.
